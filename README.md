## ğŸ¨ Customization

To customize this portfolio:

1. **Update Personal Information**
   - Modify the hero section text
   - Update about section details
   - Add your projects and experience

2. **Change Colors**
   - Primary color: `#6366f1` (Indigo)
   - Secondary color: `#ec4899` (Pink)

   - Update the projects section with your work
   - Add project images to the assets folder
   - Create modal content for each project

4. **Update Skills & Experience**
   - Modify skill categories and items
   - Add experience timeline entries
   - Update dates and descriptions
